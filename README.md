## Getting started

Install:

```bash
npm install
```

Run:

```bash
npm run dev
```

## Dependencies

- [Next.js](https://nextjs.org)
- [Radix UI](https://www.radix-ui.com)
- [Tailwind CSS](https://tailwindcss.com)
- [next-themes](https://github.com/pacocoursey/next-themes)
- [@tabler/icons-react](https://tabler.io/docs/icons/react)
